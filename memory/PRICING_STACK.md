## THE GREENLINE365 TIERED VALUE STACK

### Tier 0: The Reputation "Hook" (FREE)
**Purpose:** Lead generation and competitive pressure.
- Directory Listing: Basic business profile with "Grayed-Out" status
- Ghost Badges: All 7 industry badges in grayscale with "Locked" icons
- Limited AI: Basic "Address-Centric" search recognition (no historical data)
- **The Paywall:** Clicking any locked badge triggers $400 Website Audit invitation

### Tier 1: The Booking Foundation ($1,500/mo)
- AI Booking Agent + Dedicated Phone Line ($900 value)
- Dedicated Phone Number ($20 value)
- Knowledge Base ($75 value)
- Cal.com Setup ($500 one-time included)
- Core AI Logic: Address-centric recognition for incoming calls
- Automated Reputation Starter: General reviews (no specialized polls)
- Essential Directory Sync: Basic hours and location mirroring

### Tier 2: The Marketing Engine ($2,000/mo)
- **Everything in Tier 1**
- Poll Template Engine: Specialized QR codes for "Vibe" and "Cleanliness" badge earning ($300 value)
- Email & SMS Marketing via SendGrid/Twilio ($350 value)
- Campaign Manager & Templates ($200 value)
- AI Review Responder: Brand-Voice-consistent replies to Google + Directory reviews ($150 value)
- Content Forge: Automated Blog & Newsletter sync to directory listing
- Unified Dashboard: Coupon Engine and Analytics access

### Tier 3: The Intelligence Command ($2,500/mo)
- **Everything in Tier 1 & 2**
- Full Property Intelligence: Property Passport + "Behind the Walls" photo vault ($500 + $250 value)
- Incident & Clearance Layer: Manager "Clear" authority + Clean Bills of Health ($400 value)
- Multi-User Relationship Map: Link Spouses, Tenants, Property Managers
- Seal of Approval Eligibility: Access to "Super Featured" carousel
- White-Label Branding ($250 value)

### Complexity Add-Ons (Sliding Scale)

| Add-On Component | Additional Setup Fee | Complexity Reason |
|---|---|---|
| Additional AI Agent | +$350 per agent | Persona programming, prompt engineering, KB ingestion |
| A2P 10DLC Registration | +$150 per line | Federal filing fees, carrier compliance review |
| CRM Integration | +$250 (One-time) | API mapping, webhook testing, data-flow verification |
| Property "Deep-Seed" | +$250 per property | Manual data entry, high-res media organization |
| Custom Poll Templates | +$150 per template | Copywriting, AI logic tuning |

### Setup Fee Formula
```
Total Setup = Base_Fee(Tier) + Sum(Add-On_Setup_Fees)
```

**Example: "Nightclub Elite" Setup**
- Tier 2 Marketing Bundle: $1,000 base
- 1 Additional Sales Agent: +$350
- A2P Registration (2 lines): +$300
- 3 Custom Poll Templates (Vibe, Safety, Cleanliness): +$450
- **TOTAL SETUP: $2,100** (Plus $2,670/mo recurring)

### Closing Strategy
- Credit $400 Website Audit toward first month's bundle
- Waive $250 CRM Integration fee for same-day sign
- These are "chips" to play during the close
